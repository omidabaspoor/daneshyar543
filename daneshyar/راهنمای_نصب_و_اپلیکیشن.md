اما رم ۱ گیگ تنگ است، برای همین در نصب:

- ✅ MariaDB سبک‌سازی شده (`buffer_pool=128M`)
- ✅ OPcache روشن است تا CPU کمتر مصرف شود

> **تخمین واقع‌بینانه:** ۱۰–۲۰ کاربر همزمان روان کار می‌کنند. اگر به ~۳۰+ همزمان رسیدی،
> باید رم را به ۲ گیگ ارتقا بدهی.

# بخش الف — نصب روی سرور Ubuntu

## روش خودکار (پیشنهادی) ⭐

روی سرورِ تازه‌نصب Ubuntu، این دستورات را اجرا کن:

```bash
# 1) پروژه را روی سرور بیاور
git clone https://github.com/omidabaspoor/daneshyar655.git
cd daneshyar655

# 2) اسکریپت نصب خودکار را اجرا کن
sudo bash deploy/setup-ubuntu.sh
```

اسکریپت به‌صورت خودکار این کارها را می‌کند:
1. Swap ۲ گیگ می‌سازد
2. Nginx + PHP-FPM + MariaDB نصب می‌کند
3. دیتابیس و کاربر می‌سازد + `install.sql` را اجرا می‌کند
4. فایل `.env` را با رمز تصادفی دیتابیس می‌سازد
5. کانفیگ‌های بهینه‌شده را کپی و سرویس‌ها را ری‌استارت می‌کند
6. فایروال را تنظیم می‌کند

### بعد از نصب — ۳ کار باقی‌مانده (مهم):

```bash
# 1) کلید جدید API و رمز ادمین را وارد کن
sudo nano /var/www/daneshyar/.env
#   ↳ AI_API_KEY=sk-...کلید-جدید...
#   ↳ ADMIN_PASS=یک-رمز-قوی

# 2) بعد از تست موفق، نصب‌کننده را پاک کن
sudo rm -f /var/www/daneshyar/install.php /var/www/daneshyar/migrate.php

# 3) cron برای فعال‌سازی خودکار اشتراک‌ها
crontab -e
#   این خط را اضافه کن:
*/5 * * * * curl -s 'http://127.0.0.1/api/scheduler.php?token=dy_sch_b037ca7f8ec42c0a11ff5d0bfbed5757' >/dev/null 2>&1
```

سپس در مرورگر برو به: `http://آی‌پی‌سرور/`
پنل ادمین: `http://آی‌پی‌سرور/admin/login.php`

## روش دستی (اگر اسکریپت را نمی‌خواهی)

<details>
<summary>کلیک کن برای مراحل دستی</summary>

```bash
# Swap
sudo fallocate -l 2G /swapfile && sudo chmod 600 /swapfile
sudo mkswap /swapfile && sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab

# نصب پکیج‌ها
sudo apt update
sudo apt install -y nginx mariadb-server php8.3-fpm php8.3-mysql \
     php8.3-curl php8.3-mbstring php8.3-gd php8.3-xml php8.3-opcache

# پروژه
sudo cp -r daneshyar655 /var/www/daneshyar
sudo chown -R www-data:www-data /var/www/daneshyar
sudo chmod -R 775 /var/www/daneshyar/uploads /var/www/daneshyar/books

# دیتابیس
sudo mysql -e "CREATE DATABASE daneshyar CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
sudo mysql -e "CREATE USER 'daneshyar'@'localhost' IDENTIFIED BY 'رمز-قوی';"
sudo mysql -e "GRANT ALL ON daneshyar.* TO 'daneshyar'@'localhost'; FLUSH PRIVILEGES;"
sudo mysql daneshyar < /var/www/daneshyar/sql/install.sql

# env
sudo cp /var/www/daneshyar/.env.example /var/www/daneshyar/.env
sudo nano /var/www/daneshyar/.env   # مقادیر را وارد کن

# کانفیگ‌ها
sudo cp deploy/php/daneshyar-pool.conf /etc/php/8.3/fpm/pool.d/daneshyar.conf
sudo cp deploy/nginx/daneshyar.conf /etc/nginx/sites-available/daneshyar.conf
sudo ln -s /etc/nginx/sites-available/daneshyar.conf /etc/nginx/sites-enabled/
sudo rm -f /etc/nginx/sites-enabled/default
sudo cp deploy/mariadb/daneshyar-tune.cnf /etc/mysql/mariadb.conf.d/99-daneshyar.cnf

# ری‌استارت
sudo nginx -t && sudo systemctl restart nginx php8.3-fpm mariadb
```

# بخش ب — اپلیکیشن اندروید (APK)

اپ یک **پوسته‌ی WebView** است که سایت تو را داخل یک اپ نیتیو نشان می‌دهد.
از **استریم چت SSE، آپلود عکس، و کوکی/لاگین** پشتیبانی می‌کند.

> ⚠ چون سرورت با **IP و HTTP** کار می‌کند (بدون SSL)، اپ طوری تنظیم شده که
> ترافیک HTTP (cleartext) را مجاز کند. این اپ برای **نصب مستقیم (sideload)**
> عالی است، ولی **در گوگل‌پلی قابل انتشار نیست** (پلی HTTPS می‌خواهد).
> وقتی دامنه + SSL گرفتی، می‌توان آن را برای پلی هم آماده کرد.

## مرحله ۱: آدرس سرور را در اپ بگذار

فایل `android/app/src/main/res/values/strings.xml` را باز کن و این خط را عوض کن:

```xml
<string name="server_url">http://آی‌پی‌سرور-تو/</string>
```

## مرحله ۲: ساخت APK — دو راه

### راه A: ساخت ابری با GitHub Actions (بدون نصب چیزی) ⭐

۱. پروژه را روی گیت‌هاب خودت push کن.
۲. به تب **Actions** برو → workflow «**Build Android APK**» → دکمه **Run workflow**.
۳. بعد از ~۳ دقیقه، از بخش **Artifacts** فایل `daneshyar-debug-apk` را دانلود کن.
۴. APK داخل آن را روی گوشی نصب کن.

> فایل workflow از قبل ساخته شده: `.github/workflows/build-apk.yml`

### راه B: ساخت محلی با Android Studio

۱. **Android Studio** را نصب کن.
۲. پوشه‌ی `android/` را به‌عنوان پروژه باز کن (`Open`).
۳. صبر کن Gradle sync شود.
۴. منو: **Build → Build Bundle(s)/APK(s) → Build APK(s)**.
۵. APK اینجا ساخته می‌شود:
   `android/app/build/outputs/apk/debug/app-debug.apk`

### راه C: ساخت با خط فرمان (اگر Android SDK داری)

```bash
cd android
gradle wrapper --gradle-version 8.7   # فقط بار اول
./gradlew assembleDebug
# خروجی: app/build/outputs/apk/debug/app-debug.apk
```

## مرحله ۳: نصب روی گوشی

۱. APK را به گوشی منتقل کن.
۲. هنگام نصب، «نصب از منابع ناشناس» را اجازه بده.
۳. تمام! آیکون «دانش‌یار» روی صفحه‌ی گوشی ظاهر می‌شود.

## ویژگی‌های اپ

- ✅ آپلود عکس مسئله از گالری/دوربین
- ✅ ذخیره‌ی لاگین (کوکی/Remember-me)
